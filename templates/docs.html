How It Works

This application transforms a Substack RSS feed into a clean, readable HTML page and allows you to export the content in various formats.

1. Find a Substack Feed

You can enter the Substack's name or its full URL. The application will automatically find the correct RSS feed URL. Here are some valid inputs:

  • writebuildscale
  • writebuildscale.substack.com
  • https://writebuildscale.substack.com

2. View the Content

After submitting, the feed's content will be displayed in a clean, easy-to-read format. All posts from the feed are shown on a single page.

3. Download in Your Preferred Format

Using the "Download As..." dropdown menu, you can save the entire feed's content for offline use. The available formats are:

  • PDF: For universal viewing and printing.
  • Word (.docx): For editing in Microsoft Word or Google Docs.
  • PowerPoint (.pptx): A basic presentation with one post per slide.
  • HTML: A self-contained web page for offline browser viewing.
  • Plain Text (.txt): The raw text content with no formatting.
  • Markdown (.md): For use in Markdown editors or note-taking apps.
